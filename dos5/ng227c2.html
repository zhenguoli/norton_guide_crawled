Int 2Fh / 0600h         Get ASSIGN.COM Installed State               DOS 3.0+

    Determines whether the resident portion of the ASSIGN command has
    been loaded.

        Entry   AX = 0600h

        Return  AL = 0FFh | 00h   ASSIGN loaded | not loaded
