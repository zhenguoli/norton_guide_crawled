Int 2Fh / 4800h         Get DOSKEY.COM Installed State               DOS 5.0+

    Determines whether the resident portion of the DOSKEY program has
    been loaded.

        Entry   AX = 4800h

        Return  AL = 00h   DOSKEY not loaded
                AL > 00h   DOSKEY loaded
See Also: 4810h
