Function 5802h          Get Upper-Memory Link                        DOS 5.0+

    Specifies whether programs can allocate memory from the upper
    memory area.

        Entry   AX = 5802h

        Return  AL = 01h   If the upper-memory area is linked
                AL = 00h   Upper-memory area is not in memory chain
See Also: 5803h 48h
