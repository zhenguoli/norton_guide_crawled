Function 6520h          Convert Character                            DOS 4.0+

    Converts the specified character to an uppercase character using
    the current uppercase table.

        Entry   AX = 6520h
                DL = Character to convert

        Return  DL = Capitalized character
See Also: 6521h 6522h
