Function 440Dh /46h     Set Media ID                                 DOS 4.0+

    Sets the volume label, serial number, and file system for the
    specified drive.

        Entry   AX    = 440Dh
                BX    = Drive number (0 = default, 1 = A, etc.)
                CH    = 08h (device category, must be 8)
                CL    = 46h
                DS:DX = Pointer to a MID structure

        Return  AX = Error code, if CF is set
                     | 01h Invalid function
                     | 02h File not found
                     | 05h Access denied

    ------------------------------------------------------------------

    The MID structure contains information that uniquely identifies a
    disk or other storage medium.
                                                     
    This function returns an error value of 2 (file not found) if the
    specified drive number is invalid.
See Also: 440Dh /66h MID
