Function 5800h          Get Allocation Strategy                      DOS 3.0+

    Returns the method DOS uses to allocate memory.

        Entry   AX = 5800h

        Return  AX = allocation-strategy value

        
    Refer to function 5801h for a list of allocation-strategy values.
See Also: 48h 5801h 5802h 5803h
