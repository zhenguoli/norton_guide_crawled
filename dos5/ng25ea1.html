Int 2Fh / AD81h         Set KEYB.COM Active Code Page                DOS 3.3+

    Sets the active code page for KEYB to the specified code page.

        Entry   AX = 0AD81h
                BX = Code page ID

        Return  Nothing
                or
                AX = Error code, if CF is set
                     | 01h Invalid ID
See Also: CodePageIDs
