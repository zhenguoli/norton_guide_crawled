Int 2Fh / AD82h         Set KEYB.COM Country Flag                    DOS 3.3+

    Sets the current value of the KEYB country flag.

        Entry   AX = 0AD82h
                BL = 00h | 0FFh   Domestic | foreign keyboard

        Return  Carry set if BL not 00h or 0FFh
See Also: AD83h
