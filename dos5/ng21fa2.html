Int 2Fh / 0102h         Remove File From Print Queue                 DOS 3.0+

    Removes a specified file from the PRINT queue.

        Entry   AX    = 0102h
                DS:DX = Pointer to ASCIIZ filename (* ? allowed)

        Return  Nothing
                or
                AX = Error code, if CF is set
                     | 02h File not found
See Also: 0101h
