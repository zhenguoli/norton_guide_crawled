Int 2Fh / AD83h         Get KEYB.COM Country Flag                    DOS 5.0+

    Returns the current value of the KEYB country flag.

        Entry   AX = 0AD83h

        Return  BL = 00h | 0FFh   Domestic | foreign keyboard
See Also: AD82h
