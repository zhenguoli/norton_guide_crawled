Function 2Ah            Get Date

    Reports the current DOS system date (the date maintained by the
    clock device).

        Entry   AH = 2Ah

        Return  AL = Day of the week (0 - 6; 0 = Sunday, 1 = Monday)
                CX = Year (1980 - 2099)
                DH = Month (1 - 12)
                DL = Day (1 - 31)
See Also: 2bh 2ch 2dh
