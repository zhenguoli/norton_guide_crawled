Function 2Fh            Get Disk Transfer Address (DTA)

    Gets the current DTA (Disk Transfer Address).

        Entry   AH = 2Fh

        Return  ES:BX = Address of current DTA

    ------------------------------------------------------------------

    There is no way to determine the size of the buffer at the DTA.
See Also: 1ah 4eh
