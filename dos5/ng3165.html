Function 19h            Get Default Drive

    Returns the number of the default drive.

        Entry   AH = 19h

        Return  AL = Drive number (0 = A, 1 = B, 2 = C etc.)

    ------------------------------------------------------------------

    The drive number used by Set/Get Default Drive (functions 0Eh and
    19h) is one less than the drive number used by other functions.
See Also: 0eh 3305h 47h
