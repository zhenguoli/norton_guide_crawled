Int 2Fh / B707h         Set APPEND.EXE Modes Flag                    DOS 4.0+

    Sets the current operation modes for the APPEND command.


        Entry   AX = 0B707h
                BX = APPEND operation modes   (see B706h)

        Return  Nothing
See Also: B706h
