Int 2Fh / AD80h         Get KEYB.COM Version Number                  DOS 3.3+

    Returns the major and minor version numbers for the KEYB program.

        Entry   AX = 0AD80h

        Return  AL = 0FFh   If KEYB installed
                BL = Minor version
                BH = Major version
                ES,DI changed
