Function 1Fh            Get Default DPB                              DOS 5.0+

    Retrieves the drive parameters for the default drive.

        Entry   AH = 1Fh

        Return  AL = 00h   DS:BX = address of DPB structure
                AL = FFh   Error, invalid drive or disk error
See Also: 32h DPB
