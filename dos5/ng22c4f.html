Int 2Fh / 1400h         Get NLSFUNC.EXE Installed State              DOS 3.3+

    Determines whether the resident portion of the NLSFUNC program has
    been loaded.

        Entry   AX = 1400h

        Return  AL = 0FFh | 00h   NLSFUNC loaded | not loaded
