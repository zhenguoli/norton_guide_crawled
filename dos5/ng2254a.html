Int 2Fh / 0105h         Release Print Jobs                           DOS 3.0+

    Restarts the PRINT queue.

        Entry   AX = 0105h

        Return  Nothing


    Programs must use this function after calling Hold Print Jobs and
    Get Status (Int 2Fh function 0104h) to restart the current print
    job.
See Also: 0104h
