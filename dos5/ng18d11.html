Function 5701h          Set File Date and Time

    Sets the time and date for a file, replacing the time and date set
    for the file when it was last modified.

        Entry   AX = 5701h
                BX = File handle
                CX = Time
                DX = Date

        Return  Nothing
                or
                AX = Error code, if CF is set
                     | 06h Invalid handle

    ------------------------------------------------------------------

    The values in CX and DX have the following format:

                Bits    Contents
        CX      0-4     Seconds divided by 2
                5-10    Minutes (0-59)
                11-15   Hours (0-23 on a 24-hour clock)

        DX      0-4     Day of month (1-31)
                5-8     Month (1 = January, 2 = February, etc.)
                9-15    Year offset from 1980
                        (add 1980 to get actual year)
See Also: 5700h
