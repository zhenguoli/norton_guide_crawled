Function 6601h          Get Global Code Page                         DOS 3.3+

    Identifies the code page currently used by all programs.

        Entry   AX = 6601h

        Return  BX = Global code page (set by user)
                DX = System code page (set at startup)
                or
                AX = Error code, if CF is set
                     | 02h File not found
See Also: 6602h CodePageIDs
