Int 2Fh / B700h         Get APPEND.EXE Installed State               DOS 4.0+

    Determines whether the resident portion of the APPEND command has
    been loaded.

        Entry   AX = 0B700h

        Return  AL = 0FFh | 00h   APPEND loaded | not loaded
